Krita Data Merge Generator Manual

v1.53

About

For this plugin work to require Template (kra:krita native document file) and CSV data(a Text file)
This plugin replaces text objects and FileLayers with specific tagged.

Usage

  1. Preparation,Create a kra image for template and a CSV file for merge data
  2. Open the image
  3. Go to Tools → Scripts → Data Combine
  4. Open a dialog,and set value (Option)
  5. Select the CSV file
  6. Wait work to finish(A bit slowly)

Setting of the kra file

Top GroupLayer that named "Template",to append all resources for nessesary into the Layer.
For example, you can append layers for replace text and/or image by data merge

Top level GroupLayer : This Layer's name must be "Template"
This contain a following Layers or/and Masks

  • <<Tag>> : In the string TextElement on a VectorLayer
  • %%Tag%% : If it use for FileLayer(s)'s name (It also linked to dummy.png)
  • TransformMask : Add to for the FileLayer
  • Normal PaintLayers(Images) : Optionally
  • Normal VectorLayers(Shapes) : Optionally
For tag naming example in kra file:
  • %%IMG%% , %%PHOTO%% , %%HEADER2%%
  • <<NAME>>,<<ZIPCODE>>,<<YEAR>>
  • The person from <<CITY>>,<<ラテン文字以外のタグ>>

Setting of the CSV data

Each rows are separated with commas
Enclosing by %% for only FileLayer replace tag
For Example

%%IMG%%,TITLE,COLOR,MOOD
a.png,Harmony,Yellow and Green,Natural
b.png,IceCircle,Blue,Cold
c.png,Double square,Magenda and Green,Accent
d.png,Slash,Orange and Winter blue,Dynamic
Define tags in the 1st line.
(The name of the tag corresponds to the one used in the template image.)
The 2nd and any later,describe for corresponding data.

Dialog




Document Width/Height = Document size for output
Default document size is A4 size(300dpi:2480*3508 px)

Padding W/H = Padding for each output
BaseX/Y = Base position of output
# Recommend Padding (unit:px)
# 300dpi : 1mm = 11.8px , 5mm = 59px , 10mm = 108px , 15mm = 177px
# 600dpi : 1mm = 23.6px , 5mm = 108px , 10mm = 236px , 15mm = 344px
Edit setting.py if you want to change the default settings.
